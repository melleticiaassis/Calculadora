# 🧮 Calculadora em JavaScript

Projeto de calculadora simples desenvolvido utilizando **HTML, CSS e JavaScript**.

O objetivo do projeto é praticar lógica de programação, manipulação do DOM e criação de interfaces interativas no navegador.

## 🚀 Funcionalidades

* Operações matemáticas básicas
* Soma
* Subtração
* Multiplicação
* Divisão
* Limpar display
* Apagar último número

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 📂 Estrutura do projeto

```
calculadora-js/
│
├── index.html
├── css
│   └── style.css
└── js
    └── script.js
```

## ▶️ Como executar

1. Clone o repositório

```
git clone https://github.com/seu-usuario/calculadora-js.git
```

2. Abra o arquivo:

```
index.html
```

## 🎯 Objetivo

Este projeto foi desenvolvido como prática de **desenvolvimento front-end**, focando em lógica de programação e manipulação de eventos em JavaScript.
